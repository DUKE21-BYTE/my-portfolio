# Dennis Fundi - Portfolio Website

A premium, high-performance portfolio website built with HTML, CSS, and JavaScript. designed to showcase web development and cybersecurity skills.

## Features

- **Premium UI**: Dark mode, glassmorphism, and smooth animations.
- **Responsive**: Mobile-first design that looks great on all devices.
- **Sections**:
  - **Hero**: Strong value proposition and stats.
  - **Skills**: Categorized skills (Web, Security, Tools).
  - **Projects**: Featured projects with "tech stack" tags.
  - **Services**: clear "What I Can Build For You" section.
  - **Labs**: Experimental projects showcase.
  - **Contact**: Functional form layout and social links.

## How to Run

1.  **Open locally**: Simply double-click `index.html` to view the website in your browser.
2.  **Live Server (Recommended)**: If you use VS Code, install the "Live Server" extension and click "Go Live" for the best experience.

## Customization

- **Images**: Replace the placeholder project gradients with actual screenshots in the `img` folder (you'll need to create one if you have images).
- **Links**: Update the `href="#"` links in `index.html` to point to your real GitHub repos and live demos.
- **Email**: Update `fundidennis@example.com` in `index.html` to your actual email.

## Deploying

This is a static site, so you can host it for free on:

- **Netlify**: Drag and drop this folder onto Netlify.
- **GitHub Pages**: Push to a GitHub repository and enable Pages.
- **Vercel**: Import your repository.
